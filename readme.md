## Fonctionnement du projet

> @Require{
>    "php": 8.1,
>    "composer": 2.*
>}
>


> Pour lancer le projet, chargez l'autoloader avec la commande:
>
> ```composer install```
>
> puis insérez vos données de config de DB sur le fichier [dbConfig.php](/config/dbConfig.php)
